# Field Types

## Klaviyo List

Dropdown selection.

Selected value is stored as an associative array:

```
[
    'id' => listId,
    'name' => listName,
]
```

## Klaviyo Lists

Group checkbox for multi-selection of Klaviyo lists.

Values stored in an associative array of key/value pairs:

```
[listId => listName]
```
